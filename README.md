# osm_tile_clonner

A simple tool to clone some tiles from osm and preserve structure.

!!!!!!DO NOT USE INTENSIVELY, IT WILL OVERLOAD OSM SERVERS.!!!!!!!!

## Usage
juste change min,max lat,lon and zoom in the script and run it.
